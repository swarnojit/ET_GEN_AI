## 📌 Overview

**AI Money Mentor** is an AI-powered personal finance platform built for the **ET AI Hackathon 2026 (Problem #9)**.

It solves a critical problem:

> 95% of Indians don’t have a financial plan, while financial advisors charge ₹25,000+ per year.

Our solution provides:

* 📊 Instant tax analysis (Old vs New regime)
* 📄 AI-powered Form-16 parsing
* 💡 Personalized financial & investment recommendations
* 📈 Portfolio growth calculators (SIP & Lumpsum)

---

## 🎯 Key Features

### 🤖 AI Tax Wizard

* Upload **Form-16 / financial documents**
* Auto-extract **18+ financial fields**
* Compare **Old vs New tax regimes (FY 2025–26)**
* Get **AI-generated explanations**

---

### 📊 Portfolio Calculator

* SIP & Lumpsum investment projections
* Visual wealth growth insights
* Helps deploy tax savings intelligently

---

### 📚 Tax Education Module

* Blog-style explanation of:

  * Old vs New regime
  * Deductions (80C, 80D, etc.)
* Beginner-friendly financial literacy

---

### 💡 Smart Recommendation Engine

* Detects:

  * Missed deductions
  * Tax-saving opportunities
* Suggests investments based on:

  * Risk profile (Low / Medium / High)

## ⚡ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/your-username/ai-money-mentor.git

# Navigate to project
cd ai-money-mentor

# Install backend dependencies
pip install -r requirements.txt

# Run backend
python app.py
```
